Welcome to Virtual Reality!


First invented in 1968 by Ivan Sutherland who created a head-mounted display to project images onto a screen, the world of VR has come a very, very long way.



So what exactly is Virtual Reality (VR)?

VR is a simulated 3D environment that allows users to explore and interact in a virtual space, mimicking reality which allows users to immerse themselves in a world outside their physicality. There are six types of VR categories:

  • Non-immersive: using a computer screen and keyboard, the user is immersed inside a video game
  • Semi-immersive: partial VR experience using a computer screen, glasses or headset with no physical movement.
  • Fully immersive: incorporating sight, sound, movement, and in some instances, touch using equipment like headgear, goggles or gloves.
  • Collaborative VR: an interactive VR environment for users across the world to interact using headphones and headsets.
  • Augmented reality: virtual simulation overlaid with real world environment such as a virtual store where you can visualise goods.
  • Mixed reality (MR): users can interact with digital and physical elements in a metaverse.
